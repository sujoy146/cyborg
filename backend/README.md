# Cyborg — StegoGAN v9.1 API

Multi-layer encrypted image steganography using deep learning.

## Endpoints

- `GET /health` — Health check
- `GET /images` — Random cover images
- `POST /encode` — Hide message in image
- `POST /decode` — Extract message from stego image
